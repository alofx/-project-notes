## 6. Conversion Tracking Setup

### Must-Have Conversions

1. **Phone Calls from Ads** (Google automatically tracks call extension clicks)
2. **Phone Calls from Website** (use Google forwarding number)
3. **Form Submissions** (book now form on site)
4. **Website Clicks to Call** (tel:305-853-6101 clicks)

### Google Ads Conversion Tag Installation

Add this to `<head>` of every page on vice-details.com:

```html
<!-- Google tag (gtag.js) -->
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-XXXXXXXXX"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-XXXXXXXXX');
</script>
```

> Replace `AW-XXXXXXXXX` with your actual Google Ads conversion ID after account creation.

### Form Submission Conversion Event

Add this script that fires when booking form is successfully submitted:

```html
<script>
gtag('event', 'conversion', {
    'send_to': 'AW-XXXXXXXXX/YYYYYYYYYYYY',
    'value': 165.0,
    'currency': 'USD'
});
</script>
```

### Call Conversion Event

For clicks on phone number links:

```javascript
document.querySelectorAll('a[href^="tel:"]').forEach(link => {
  link.addEventListener('click', () => {
    gtag('event', 'conversion', {
      'send_to': 'AW-XXXXXXXXX/ZZZZZZZZZZZZ',
      'value': 165.0,
      'currency': 'USD'
    });
  });
});
```

## 9. Weekly Optimization Routine

**Every Monday (15 min):**
1. Check spend vs. budget pacing
2. Review Search Terms report → add new negative keywords
3. Check Quality Score — improve low QS keywords (add to ad copy)
4. Pause keywords with CPA > $100 and no conversions in 7 days
5. Add new keywords from Search Terms that converted
6. Check ad strength — add more headlines to "Poor" or "Average" ads
7. Review competitor auction insights

**Every Month (30 min):**
1. Full keyword performance review
2. Refresh ad copy (test new angles)
3. Expand to new locations if ROI is strong
4. Increase budget on winning campaigns by 20%
5. Launch Performance Max campaign if ready
6. Update landing page based on ad performance data
