## Ad Creative for Meta

### Creative Strategy for Car Detailing

Meta is VISUAL. Your before/after content will be your #1 performer. Here's the ad framework:

**Video Ad Structure (15-30 seconds):**
1. **Hook (0-3 sec):** "This car looked brand new after 3 hours."
2. **Before (3-8 sec):** Show dirty interior/exterior
3. **Process (8-15 sec):** Quick cuts of foam, scrubbing, polishing
4. **After (15-20 sec):** Shiny, clean car reveal
5. **CTA (20-25 sec):** "Book your detail — we come to you. Link in bio."

**Image Ad Structure:**
- Split-screen before/after
- Bold text overlay: "From $165 — We Come To You"
- Vice pink/cyan brand colors
- Phone number or "Book Now" button

## Meta Pixel Setup

### Install Pixel on Website

Add this to `<head>` of every page:

```html
<!-- Meta Pixel Code -->
<script>
!function(f,b,e,v,n,t,s)
{if(f.fbq)return;n=f.fbq=function(){n.callMethod?
n.callMethod.apply(n,arguments):n.queue.push(arguments)};
if(!f._fbq)f._fbq=n;n.push=n;n.loaded=!0;n.version='2.0';
n.queue=[];t=b.createElement(e);t.async=!0;
t.src=v;s=b.getElementsByTagName(e)[0];
s.parentNode.insertBefore(t,s)}(window, document,'script',
'https://connect.facebook.net/en_US/fbevents.js');
fbq('init', 'YOUR_PIXEL_ID');
fbq('track', 'PageView');
</script>
<noscript>
<img height="1" width="1" style="display:none" 
src="https://www.facebook.com/tr?id=YOUR_PIXEL_ID&ev=PageView&noscript=1"/>
</noscript>
<!-- End Meta Pixel Code -->
```

### Standard Events to Track

```javascript
// Lead form submission
fbq('track', 'Lead');

// Phone number click
fbq('track', 'Contact');

// Booking page view
fbq('track', 'ViewContent');

// Initiate checkout / booking started
fbq('track', 'InitiateCheckout');

// Purchase / booking confirmed
fbq('track', 'Purchase', {
  value: 165.00,
  currency: 'USD'
});
```

## Weekly Meta Optimization

**Every Monday:**
1. Review Cost Per Lead by ad set — pause any >$40
2. Review CTR by creative — pause any <1%
3. Check frequency — if >3, refresh creative
4. Review comments — respond to all, hide spam
5. Check placement breakdown — shift budget to best performers
6. Review audience breakdown — age, gender, platform

**Every 2 Weeks:**
1. Test 1-2 new creative angles
2. Test new audience interests
3. Update ad copy based on top-performing messages
4. Expand winning ad sets by 20% budget

## Comparison: Google Ads vs Meta Ads

| Factor | Google Ads | Meta Ads |
|--------|-----------|----------|
| **Intent** | High (actively searching) | Lower (scrolling, discovery) |
| **CPC** | $4-8 | $1.50-3.00 |
| **Cost Per Lead** | $35-60 | $15-35 |
| **Best For** | Capturing demand | Creating demand |
| **Visual** | Text ads + extensions | Video, carousel, images |
| **Speed to Lead** | Immediate (click to call) | Form fill, then follow up |
| **Spanish Market** | Good | Excellent |
| **Retargeting** | Basic | Excellent |
| **Creative Work** | Low | High |

**Recommendation:** Run BOTH. Google for people searching "car detailing near me." Meta for people who don't know they need detailing until they see your before/after video.

**Budget Split ($750/month each):**
- Google: $750/month — capture existing demand
- Meta: $750/month — create new demand

Or alternate: Month 1 Google only, Month 2 add Meta, Month 3 run both.
